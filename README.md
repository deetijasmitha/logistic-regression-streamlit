
# Logistic Regression Web Application

This project is an interactive web application that demonstrates the use of a logistic regression model through a simple and user-friendly interface. The application allows users to input data values and instantly receive predictions, making it easy to understand how classification models work in real-world scenarios.
The focus of this project is on clarity, usability, and practical implementation. It is suitable for learning purposes, academic projects, and portfolio demonstration.

## Features
* Interactive web-based user interface
* Real-time prediction based on user input
* Clean and simple design for easy understanding
* Demonstrates end-to-end machine learning workflow

## How to Run the Project
# 1. Clone the repository
bash
git clone https://github.com/deetijasmitha/logistic-regression-streamlit.git
cd logistic-regression-streamlit
# 2. Install dependencies
bash
pip install -r requirements.txt
# 3. Run the application
bash
streamlit run app.py
